# MOODS DE-T8 Beta Releases

Public APK assets and the stable update manifest for the permanently separate MOODS DE-T8 Beta Android application.

- Application ID: `com.moodsnext.det8`
- Release channel: `beta`
- Manifest: [`beta/update.json`](beta/update.json) after the first permanent prerelease
- Tags: `det8-beta-vX.Y.Z`

This repository contains no application source, credentials, keystores or private implementation documentation. Every release is a prerelease. Android may require **Install unknown apps** permission and always requires the user to approve sideload installation.

Physical provider playback remains unproven until explicitly recorded in release notes.
